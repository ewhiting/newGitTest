Git Command Line:
Bash-style commands
git clone https://github.com/ewhiting/repoName.git


cd /Directory/Where/You/Want/To/Exist
git status                            -check the status of the current directory (must be in the dir via commandline)
git pull                              -update local branch/directory


git add FileName.txt                  -add named file to list of files
git add -A


git commit -m
git push
